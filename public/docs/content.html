¿Qué es PickPad?

PickPad es un sistema integral de gestión electoral y fiscalización diseñado para facilitar el control, monitoreo y análisis de procesos electorales. La aplicación permite a organizaciones políticas gestionar padrones electorales, coordinar fiscales de mesa, realizar seguimiento de votantes y obtener estadísticas en tiempo real durante la jornada electoral.

Objetivos del Sistema

  • Gestión Centralizada: Administrar toda la información electoral desde una plataforma unificada
  • Fiscalización Eficiente: Permitir a los fiscales registrar la asistencia de votantes en tiempo real
  • Seguimiento Personalizado: Identificar votantes clave mediante el sistema de "picks"
  • Análisis en Tiempo Real: Generar estadísticas y proyecciones de participación durante la jornada
  • Coordinación de Equipos: Facilitar la comunicación y asignación de recursos entre diferentes roles

Características Principales

Multiplataforma

Funciona en navegadores web y puede instalarse como aplicación móvil (PWA) en dispositivos Android e iOS.

Acceso Offline Limitado

Una vez instalada la PWA, ciertos recursos permanecen disponibles sin conexión activa.

Sincronización Automática

Los datos se actualizan en tiempo real para todos los usuarios conectados simultáneamente.

Control de Acceso

Sistema de 5 niveles de permisos que determina qué funcionalidades ve cada usuario.

Importación Masiva

Cargue padrones completos desde archivos Excel o CSV con validación automática de datos.

Exportación Flexible

Descargue datos filtrados en formato Excel para análisis externo o respaldos.

Tecnología

PickPad está construido con tecnologías web modernas:

  • Frontend: React 18 con Vite para una experiencia de usuario rápida y fluida
  • Base de Datos: Supabase (PostgreSQL) con políticas de seguridad a nivel de fila
  • Diseño: Tailwind CSS para una interfaz responsiva y moderna
  • Visualización: Chart.js para gráficos interactivos de estadísticas
  • PWA: Service Workers para funcionalidad offline y experiencia de aplicación nativa

Roles y Permisos

PickPad implementa un sistema jerárquico de 5 niveles de usuario, donde cada nivel hereda los permisos de los niveles superiores. El número más bajo indica mayor nivel de acceso.

Jerarquía de Roles

Nivel Rol Descripción Usuarios Típicos
1 Super Administrador Acceso total al sistema sin restricciones Desarrolladores, Coordinadores Generales
2 Administrador Gestión completa de usuarios, configuración y datos Jefes de Campaña, Responsables de Operaciones
3 Coordinador Supervisión de fiscales y acceso a estadísticas Coordinadores de Circuito, Referentes Zonales
4 Fiscal Fiscalización de mesa asignada y consultas Fiscales de Mesa, Observadores
5 Solo Lectura Consulta del padrón sin capacidad de modificación Colaboradores, Personal de Apoyo

Importante: Principio de Mínimo Privilegio

Asigne a cada usuario el nivel mínimo de permisos necesario para cumplir su función. Esto reduce riesgos de seguridad y errores operativos.

Restricciones Especiales

Edición de Picks

  • Niveles 1-2: Pueden editar cualquier pick sin restricciones
  • Nivel 3: Pueden editar picks no verificados de cualquier usuario
  • Nivel 4: Solo pueden editar sus propios picks no verificados

Fiscalización

  • Niveles 1-3: Pueden fiscalizar cualquier mesa del sistema
  • Nivel 4: Solo pueden fiscalizar la mesa que les fue asignada explícitamente

Visualización de Estadísticas

  • Niveles 1-3: Acceso completo a todas las estadísticas y reportes
  • Niveles 4-5: Sin acceso al módulo de estadísticas
